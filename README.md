# pawa-command-init

PAWA 命令行init插件，工程初始化工具

## Install
放在pawa模块下的node_modules目录中，例如：nodejs\node_modules\pawa\node_modules\pawa-command-init

## Usage
    
pawa init -h

    Usage: init <name> [options]

    Options:
    
      -h, --help         output usage information
      -s, --spa          spa app

创建一个SPA应用工程  

```shell
pawa init projectName --spa   
```
